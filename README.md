# RedMaple

RedMaple is a branch of `Scut` game engine, upgraded to .Net Core platform.

## Requirements

On .Net Core 1.0 platform

## Features

* Use dotnet core framework;
* News database storage library;
* MongoDB database supported;
* News RPC library;
